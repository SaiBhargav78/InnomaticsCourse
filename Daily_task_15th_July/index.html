If-Else Conditional Statements

  • Write a program that checks if a number is positive, negative, or zero.
  • Print the appropriate message for each case using if-else statements.

......

// If-Else Conditional Statements
function checksign(){
let num = Number(document.getElementById("num").value);
if(num>0){
document.getElementById("result1").innerHTML = "Positive";
}
else if(num === 0){
document.getElementById("result1").innerHTML = "Zero";
}
else{
document.getElementById("result1").innerHTML = "Negative";
}
}

......

For Loop

  • Write a program that prints the numbers from 1 to 10 using a for loop.

......

// For Loop
function onetoten(){
document.getElementById("result4").innerHTML = "";
for(var i = 1; i<11; i++){
document.getElementById("result4").innerHTML += i + " ";
}
document.getElementById("result4").innerHTML += 10;
}


Reverse For Loop

  • Write a program that prints the numbers from 10 to 1 using a for loop.

......

// Reverse For Loop
function tentoone(){
document.getElementById("result5").innerHTML = "";
for(var i = 10; i>1; i--){
document.getElementById("result5").innerHTML += i + " ";
}
document.getElementById("result5").innerHTML += 1;
}
